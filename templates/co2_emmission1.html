CO2 Prediction

A Machine Learning Model to predict the CO2 Emission from the Vehicle

Abstract

The problem statement was to build and train a ML model by using the provided data, which will further predict the co2 emmission based on input data. As the data to be predected is a regression type of data, here as a primary approach we have build a model by using Linear Regression Algorithm. While build the model we have used different EDA, Feature Engineering and Feature selection techniques to improve the predection accuracy. The model is build by using Linear Regression Algorithm and decision tree algorithm. currently the Linear Regression model performance wrt r2_socre is above 0.92 for testing dataset as well as on training data set, whearas performance for Decision Tree model wrt r2_socre is above 0.99 for testing dataset as well as on training data set. Also the values for MSE and MAE are low and prety close to each other.

Understanding the Data
Make : Manufacturer of the vehicle
Model :
4WD/4X4 = Four-wheel drive
AWD = All-wheel drive
FFV = Flexible-fuel vehicle
SWB = Short wheelbase
LWB = Long wheelbase
EWB = Extended wheelbase

Transmission :
A = automatic
AM = automated manual
AS = automatic with select shift
AV = continuously variable
M = manual
3 - 10 = Number of gears

Fuel type :
X = regular gasoline
Z = premium gasoline
D = diesel
E = ethanol (E85)
N = natural gas

Fuel consumption City and highway :
fuel consumption ratings are shown in
litres per 100 kilometres (L/100 km) - the combined rating (55% city, 45% hwy) is shown in L/100 km and in miles per imperial gallon (mpg)

CO2 emissions :
the tailpipe emissions of carbon dioxide (in grams per kilometre) for combined city and highway driving
